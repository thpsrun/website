Qualifying Information

  • All runners are required to join the THPS Speedrun Discord and agree to the community rules. No exceptions.
  • PLEASE make sure your Discord name in this server is the same (or similar enough to) what is on here!
  • 6 maps will be selected from the pool before qualifying begins, excluding newly introduced maps:
    • THPS3: Foundry + 2 Random
    • THPS4: College + 2 Random
  • Tournament will use the Points system seen on this site to determine seeding. Tie-breakers are, in order:
    1. Average placement
    2. Most WRs
    3. Most second places
    4. Most third places
    5. Coin Flip
  • [!] Runners will need to input a tag in the description of the submission for their run to count.
    • This is to easily distinguish between who doesn't want to play and who does.
    • If you submit a run, but forget to include the tag or would like to submit it after approval; lemme know!
  • A special page will be shown on this site to display up-to-date/approved runs, separate from the real THPS3+4 boards.

Tournament Rules will be shown closer to the event, but it is pretty much the same as THPS1+2's except for map striking.
